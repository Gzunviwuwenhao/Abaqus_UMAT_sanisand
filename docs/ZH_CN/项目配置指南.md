### 项目的配置指南

### 1. Vscode 配置

#### 1.1 安装 Modern Fortran 插件

如果读者使用Vscode打开该项目, 首先在扩展·商店中安装Modern Fortran, 在工作区的setting.json设置编译器路径，例如gfotran编译器可以到([github链接](https://github.com/niXman/mingw-builds-binaries/releases).)下载,解压后将bin目录下的gfortran.exe路径填入setting.json中，例如:

``` json
{
  "fortran.linter.compilerPath": "D:/mingw64/bin/gfortran.exe",
  "fortran.linter.compiler": "gfortran"
}
```

#### 1.2 配置模块的生成路径和输出路径

```json
{
  "fortran.linter.modOutput": "${workspaceFolder}/include",
    "fortran.linter.includePaths": [
    "${workspaceFolder}/include"
  ]
}
```

#### 1.3 代码的格式化工具

```json
{
  "fortran.formatting.formatter": "fprettify",
  "fortran.formatting.fprettifyArgs": [
    "-c",
    "${workspaceFolder}/fprettify.cfg"
  ]
}
```

最后工作区的setting.json文件内容大致如下:

```json
{
  "fortran.linter.compilerPath": "D:/mingw64/bin/gfortran.exe", // 编译器路径
  "fortran.linter.compiler": "gfortran", // 编译器类型
  "fortran.linter.modOutput": "${workspaceFolder}/include", // 模块路径输出
  "fortran.linter.includePaths": [
    "${workspaceFolder}/include"  // 模块包含路径
  ],
  "fortran.formatting.formatter": "fprettify", // 代码格式化工具
  "fortran.formatting.fprettifyArgs": [
    "-c",
    "${workspaceFolder}/fprettify.cfg" // 代码格式化配置文件
  ]
}
```
